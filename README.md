# Pilates Marketplace — Design Prototype

Mobile-first design exploration for a two-sided Pilates booking marketplace.
Vite + React + TypeScript + Tailwind v4 + lucide-react.

Nine screens, one phone-frame canvas, screen-switcher at the top:
Onboarding · Discover · Studio · Instructor · Booking · My bookings · Search · Profile · Instructor dashboard.

## Run locally

```bash
npm install
VITE_BASE=/ npm run dev
```

Open http://localhost:5173.

## Deploy

A GitHub Actions workflow builds and publishes to GitHub Pages on push to `main`.

## Aesthetic

Editorial wellness, warm neutrals, generous whitespace. Bone (`#FAF7F2`) surface,
warm-charcoal ink (`#1F1B16`), terracotta CTAs (`#C97B5B`), sage accents
(`#8A9A7B`). Fraunces serif headlines paired with Inter body. No glassmorphism,
no neon gradients, no purple-blue tech vibes.
